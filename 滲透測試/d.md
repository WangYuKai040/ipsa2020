# MAC位址
```
區域網路位址（LAN Address），乙太網路位址（Ethernet Address）或實體位址（Physical Address）
它是一個用來確認網路裝置位置的位址。在OSI模型中，第三層網路層負責IP位址，第二層資料鏈結層則負責MAC位址
```
